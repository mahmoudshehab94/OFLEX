# دليل المستخدم: حل مشكلة الإشعارات

## المشكلة

تظهر رسالة خطأ عند تفعيل الإشعارات:
```
غير مُفعّل. يُرجى التواصل مع الإدارة OneSignal
```

---

## الحل السريع (3 خطوات)

### 1️⃣ عطّل مانع الإعلانات

إذا كان لديك AdBlock أو uBlock Origin أو أي مانع إعلانات:

**في Chrome/Edge:**
```
1. اضغط على أيقونة المانع في شريط الأدوات
2. اختر "Pause on this site" أو "Disable for this site"
3. أعد تحميل الصفحة (F5)
```

**في Firefox:**
```
1. اضغط على أيقونة المانع
2. اختر "Disable for this site"
3. أعد تحميل الصفحة
```

---

### 2️⃣ اسمح بالإشعارات في المتصفح

**في Chrome/Edge:**
```
1. اضغط على أيقونة القفل 🔒 بجانب عنوان الموقع
2. اذهب إلى "Site settings" (إعدادات الموقع)
3. ابحث عن "Notifications" (الإشعارات)
4. غيّرها إلى "Allow" (سماح)
5. أعد تحميل الصفحة
```

**في Firefox:**
```
1. اضغط على أيقونة القفل 🔒
2. اختر "Connection secure" → "More information"
3. علامة التبويب "Permissions"
4. ابحث عن "Receive notifications"
5. قم بإلغاء تحديد "Block" واختر "Allow"
6. أغلق النافذة وأعد تحميل الصفحة
```

**في Safari (Mac):**
```
1. Safari → Preferences (أو Settings)
2. اذهب إلى "Websites"
3. اختر "Notifications"
4. ابحث عن الموقع في القائمة
5. غيّر الإعداد إلى "Allow"
```

---

### 3️⃣ جرّب التفعيل مرة أخرى

```
1. اذهب إلى تبويب "Benachrichtigungen" أو "Einstellungen"
2. اضغط على زر "تفعيل" 🔔
3. عندما يطلب المتصفح الإذن، اختر "Allow" أو "السماح"
4. انتظر رسالة النجاح
```

---

## إذا لم يعمل بعد...

### جرّب متصفح آخر

أحياناً المشكلة تكون في إعدادات المتصفح:

```
✅ Chrome → جرّب Edge أو Firefox
✅ Firefox → جرّب Chrome
✅ Safari → جرّب Chrome (يعمل أفضل)
```

### امسح Cache المتصفح

```
1. اضغط Ctrl+Shift+Delete (أو Cmd+Shift+Delete على Mac)
2. اختر "Cached images and files"
3. اختر "All time" أو "كل الوقت"
4. اضغط "Clear data" أو "مسح البيانات"
5. أغلق وأعد فتح المتصفح
```

### جرّب وضع التصفح الخاص

هذا يتجاوز جميع الإضافات والإعدادات:

```
Chrome/Edge: Ctrl+Shift+N
Firefox: Ctrl+Shift+P
Safari: Cmd+Shift+N
```

ثم افتح التطبيق وجرّب التفعيل.

## تحقق من نجاح التفعيل

عند النجاح، يجب أن ترى:

```
✅ رسالة خضراء: "تم تفعيل الإشعارات بنجاح!"
✅ زر "إيقاف" باللون الأحمر (بدلاً من "تفعيل")
✅ ملاحظة زرقاء توضح أوقات التذكيرات
✅ خيار "إعدادات متقدمة" يظهر أسفل
```

---

## متى سأحصل على التذكيرات؟

الإعدادات الافتراضية:

```
⏰ وقت البدء: 6 مساءً (18:00)
🔄 التكرار: كل 30 دقيقة
📅 أيام العطل: لا إشعارات (السبت والأحد)
⏹️ التوقف: عندما تسجل العمل أو ينتهي اليوم
```

يمكنك تغيير هذه الإعدادات من "إعدادات متقدمة".

---

## الأسئلة الشائعة

### س: لماذا لا أحصل على إشعارات؟

**تأكد من:**
1. ✅ تم تفعيل الإشعارات (زر "إيقاف" ظاهر)
2. ✅ لم تسجل العمل اليوم بعد
3. ✅ الوقت بعد 6 مساءً
4. ✅ ليس يوم سبت أو أحد (إلا إذا غيّرت الإعدادات)
5. ✅ المتصفح مفتوح (أو الإشعارات مفعّلة في النظام)

### س: هل يعمل على الهاتف؟

نعم!
- ✅ Android: يعمل في Chrome و Firefox
- ⚠️ iOS: يعمل لكن بقيود (استخدم Chrome للحصول على أفضل تجربة)

### س: هل أحتاج إلى تطبيق منفصل؟

لا، الإشعارات تعمل مباشرة من المتصفح. لا حاجة لتثبيت أي شيء.

### س: كم مرة سأحصل على التذكيرات؟

حسب إعداداتك:
- الافتراضي: كل 30 دقيقة
- يمكنك تغييره: 15، 30، 45، 60، 90، أو 120 دقيقة

### س: كيف أوقف الإشعارات؟

```
1. اذهب إلى تبويب "Benachrichtigungen"
2. اضغط على زر "إيقاف" 🔕 (الأحمر)
3. ستتوقف الإشعارات فوراً
```

### س: هل تستنزف البطارية؟

لا، Web Push Notifications خفيفة جداً ولا تؤثر على البطارية بشكل ملحوظ.

## نصائح للحصول على أفضل تجربة

### 1. استخدم Chrome أو Edge
```
✅ أفضل دعم للإشعارات
✅ أقل مشاكل
✅ يعمل على جميع الأنظمة
```

### 2. لا تغلق المتصفح تماماً
```
ℹ️ المتصفح يمكن أن يعمل في الخلفية لاستقبال الإشعارات
ℹ️ على Windows: المتصفح يبقى في System Tray
ℹ️ على Mac: المتصفح يبقى في Dock
```

### 3. تأكد من إعدادات النظام
```
Windows 10/11:
Settings → System → Notifications → Make sure browser notifications are ON

Mac:
System Preferences → Notifications → Chrome/Firefox → Allow notifications

Android:
Settings → Apps → Chrome → Notifications → Allow
```

## اتصل بالدعم

إذا جربت كل ما سبق ولم يعمل:

📧 أرسل التفاصيل التالية:
```
1. نوع المتصفح والإصدار (مثلاً: Chrome 120)
2. نوع الجهاز (Windows, Mac, Android, iOS)
3. لقطة شاشة للخطأ
4. هل مانع الإعلانات معطّل؟
5. ماذا يظهر في Console (F12 → Console)
```

---

## الخلاصة

✅ **معظم المشاكل تُحل بـ:**
1. تعطيل مانع الإعلانات
2. السماح بالإشعارات في إعدادات المتصفح
3. مسح Cache وإعادة المحاولة

❌ **لن يعمل على:**
- iOS Safari (استخدم Chrome بدلاً منه)
- المتصفحات القديمة جداً
- مواقع HTTP (يتطلب HTTPS)

🎯 **للحصول على أفضل تجربة:**
- استخدم Chrome أو Edge
- عطّل مانع الإعلانات للموقع
- اترك المتصفح مفتوحاً في الخلفية
